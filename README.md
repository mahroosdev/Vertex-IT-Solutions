# 💻 Vertex IT Solutions

**Vertex IT Solutions** is a modern, responsive single-page IT solutions website built with **HTML, CSS, JavaScript, and Bootstrap 5**.

The website presents professional technology services through a clean interface that works across desktop, tablet, and mobile devices.

## ✨ Features

- Responsive navigation bar
- Professional hero section
- Call-to-action buttons
- IT service cards
- About and approach section
- Why Choose Us section
- Contact call-to-action
- Responsive footer
- Mobile-friendly layout
- Bootstrap-based components and grid system
- Google Fonts integration

## 📑 Website Sections

The website includes:

1. Navigation bar
2. Hero section
3. Services section
4. About section
5. Our Approach section
6. Why Choose Us section
7. Contact call-to-action
8. Footer

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in a modern web browser.

No installation or development server is required.

> An internet connection is required because the project uses Bootstrap and Google Fonts through CDN links.
